# Digital Canvas - Сайт-визитка дизайнера

![Digital Canvas Preview](https://via.placeholder.com/800x400?text=Digital+Canvas+Preview)

## Описание
Digital Canvas - это современный сайт-визитка для дизайнера, демонстрирующий портфолио работ, услуги и контактную информацию. Сайт полностью адаптивный и работает на чистом HTML, CSS и JavaScript без использования фреймворков.

## Функциональность
- [x] Адаптивный дизайн для всех устройств
- [x] Темная/светлая тема с сохранением в localStorage
- [x] Навигация между страницами
- [x] Фильтрация работ в портфолио
- [x] Анимация счетчиков для статистики
- [x] Отслеживание истории посещений (localStorage)
- [x] Форма обратной связи с сохранением сообщений (localStorage)
- [x] Плавные анимации при скролле
